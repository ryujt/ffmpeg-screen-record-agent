# 1차 일정 예측 보고서

::: warning
* 일정 예측 산출식 = a * (3W + 4E + B) / 8
  * W: 최악의 경우 걸리는 시간
  * E: 일반적인 예측 시간
  * B: 최상의 경우 걸리는 시간
  * a: 분석, 설계, 테스트 및 수정 보완에 대한 예측 보정
    * 본 프로젝트는 불확실성이 낮다고 판단하여 2를 적용
* 개발 초기의 일정예측은 정확도가 상당히 낮은 점을 유의하시기 바랍니다.
:::
